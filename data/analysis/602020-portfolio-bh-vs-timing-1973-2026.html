Methodology

Portfolio construction: 60% S&P 500, 20% 20-year Treasury bond, 20% gold. Rebalanced annually at the start of each January. Between rebalances weights drift with market returns.

Timing rule: Each leg independently timed via its own 10-month SMA. Price > SMA → invested in that asset. Price ≤ SMA → that leg moves to T-bills (BIL). Signal at close of month t−1, applied to return of month t (1-month lag, no look-ahead).

Warm-up & base date: SMA seeded using monthly prices from Jan 1963 onwards. For the 1973 study, the first return (Jan 1973) uses the Dec 1972 → Jan 1973 price change, so no return month is discarded. Signal for Jan 1973 is based on Dec 1972 price vs SMA.

Annual rebalance: Notional leg values reset to 60/20/20 each January. The timing signal within each leg is unaffected by rebalancing.

Bond return model: FRED DGS20 yield-derived total return with convexity correction. Spliced with TLT ETF from 2002.

Gold: Kitco/GitHub spot price from 1963. Spliced with GLD ETF from Nov 2004.

No transaction costs, taxes or leverage.
