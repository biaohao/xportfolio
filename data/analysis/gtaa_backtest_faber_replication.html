GTAA Backtest — Faber SMA-10 Monthly Timing Model

Refreshed data · corrected bond duration proxy · splice boundary fix applied

Method: Mebane Faber GTAA (2007/2013) — monthly SMA-10 timing. Each month: if price > 10-month SMA → hold asset; else → T-bill cash. Equal weight across all assets in universe. Rebalance at month-end. Sharpe = excess return over actual T-bill return (dynamic rf).
Data fixes applied: (1) Splice boundary bug fixed — splice-month return now real, not zeroed. (2) Bond duration corrected to yield-derived par-bond formula D_mod=(1−(1+y/2)^−20)/y, eliminating ~11pp overstatement of 1982 bond bull run.
Faber (2007 paper / 2013 update) GTAA-5 reported — 1973–2012:  B&H: CAGR ~9.7%, Vol ~12%, Sharpe ~0.33, MaxDD ~−46%  |  Timing: CAGR ~10.5%, Vol ~6.8%, Sharpe ~0.74, MaxDD ~−10%, ~70% invested  |  T-Bill: CAGR ~5.0%.
⚠ Data gap: Faber used proprietary MSCI EAFE, GSCI, and NAREIT index histories from 1973. Our EFA/GSG/VNQ are ETF-only (from 2001/2006/2004) — the GTAA-5 sections below start Aug 2006 when all five ETFs have data. The SPY+IEF and SPY+IEF+GLD sections use long-history proxies back to 1962/1973.

Key Observations
  1. Drawdown reduction is the dominant benefit. Timing cuts max drawdown by ~50–75% in every window: GTAA-5 B&H −46.6% → Timing −10.6%; SPY+IEF B&H −23.1% → Timing −13.1%.
  2. CAGR advantage is modest or zero in recent decades. 2006–2025 GTAA-5 Timing (4.75%) slightly trails B&H (5.00%). The trend-following CAGR premium was primarily earned in the 1970s–1980s high-inflation, high-volatility regime.
  3. Gold improves both portfolios. Adding GLD raises 3-asset Timing Sharpe from 0.734→0.791 (long history) and GTAA-5 Sharpe from 0.514→0.640, with lower drawdowns in both cases.
  4. Sharpe matches Faber closely where data overlaps. SPY+IEF Timing Sharpe 0.734 vs Faber's reported ~0.74. Timing consistently nearly doubles Sharpe vs B&H.
  5. ~63–73% invested is typical — Faber reported ~70%. GTAA-5 averages ~63% (more assets means more individual signals can be in cash); SPY-only timing is ~74%.
  6. Bond proxy quality improved. Corrected D_mod reduces the 1982 bond return from +49.8% to +38.3%, bringing the long-history IEF proxy in line with realistic bond fund behavior.
